# Day 5: Express.js Basics

- Create routes (GET, POST)

## Mini Task
API returning a list of students (JSON)

## Files

- `package.json` - Dependencies

## How to Run
```bash
npm install
npm start
```

## API Endpoints
- GET /students - List all students
- GET /students/:id - Get student by ID
- POST /students - Add new student
