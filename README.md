## 📋 Overview
This repository contains a complete 21-day full-stack development journey covering JavaScript fundamentals, Node.js backend development, React frontend development, and advanced project building. Each day includes hands-on coding exercises, mini-projects, and real-world applications.

## 🎯 Learning Objectives
- 💻 Master JavaScript fundamentals and ES6+ features
- 🖥️ Build robust backend APIs with Node.js and Express.js
- ⚛️ Create dynamic frontends with React and modern tooling
- 🔗 Integrate frontend and backend for full-stack applications
- 📊 Work with databases and authentication systems
- 🌟 Build impressive portfolio projects

## 🚀 Quick Start Guide

### Prerequisites
Make sure you have the following installed:
- 📦 [Node.js](https://nodejs.org/) (v16 or higher)
- 📂 [Git](https://git-scm.com/)
- 💻 Code editor (VS Code recommended)

### Clone & Setup
```bash
# Clone the repository
git clone https://github.com/divyanshu-007d/rudra.git

# Navigate to project directory
cd rudra

# Choose any day to explore (example: Day 1)
cd day1

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Running Different Project Types

#### 📝 JavaScript Projects (Days 1-3)
```bash
cd dayX
node script-name.js
```

#### 🖥️ Backend Projects (Days 4-5, 11, 13-14)
```bash
cd dayX
npm install
npm start
# Server runs on http://localhost:3001
```

#### ⚛️ React Projects (Days 6-10, 12, 15-21)
```bash
cd dayX
npm install
npm run dev
# App runs on http://localhost:5173
```
